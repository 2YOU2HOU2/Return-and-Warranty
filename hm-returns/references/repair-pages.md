# H&M Return Reference

## Return URLs

| Resource | URL |
|----------|-----|
| H&M Returns | `https://www2.hm.com/en_us/returns.html` |
| H&M Account | `https://www.hm.com/us/en/account` |
| Store Locator | `https://www.hm.com/us/en/store-locator` |
| USPS/UPS/FedEx Drop-off | Any location |

## Key Notes

- H&M provides **free prepaid labels** (USPS, UPS, or FedEx)
- In-store return available at any H&M US store
- 30-day return window from delivery date
- Must have order confirmation for in-store returns
- Lingerie, swimwear, underwear non-returnable
- WAF/Cloudflare blocks headless browsers — manual mode required
