# Zara Return Reference

## Return URLs

| Resource | URL |
|----------|-----|
| Zara Returns | `https://www.zara.com/us/en/returns` |
| Zara Account | `https://www.zara.com/us/en/account.html` |
| Store Locator | `https://www.zara.com/us/en/store-locator` |
| UPS Drop-off | Any UPS location |

## Key Notes

- Zara provides **free prepaid UPS labels** for all returns
- In-store return available at any Zara US store
- 30-day return window from delivery date
- Must have order confirmation for in-store returns
- Final sale items non-returnable
- WAF blocks headless browsers — manual mode required
